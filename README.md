# Ally Programming Language
## CHANGELOG
## Syntax
### 基本文法
#### 関数宣言
```ally
fn test(){}
```
- 最小の関数です
```ally
fn test() = int {
    return 5
}
```
- 返り値を渡すこともできます
```ally
fn test(a: int, b: int) = int {
    return a + b
}
```
- 引数を取り、値を返すこともできます
#### 演算
||||
|---|---|---|
|記号|結果|計算量|
|`+`|足し算|O(1)|
|`-`|引き算|O(1)|
|`*`|掛け算|O(1)|
|`/`|割り算|O(1)|
|`<`|比較(LT)|O(1)|
|`>`|比較(GT)|O(1)|
|`<=`|比較(LTE)|O(1)|
|`>=`|比較(GTE)|O(1)|
|`==`|比較(EQ)|O(1)|
|`!=`|比較(NEQ)|O(1)|
|`&&`|論理和|O(1)|
|`||`|論理積|O(1)|
|`!`|否定|O(1)|
- 比較演算のみ、1 < 5 < 19のように3こ以上項を作ることができます
- 足し算などは1 + 1などのように2つのみ項を作ります
- ()を入れることで優先順位を変えることができます((1 + 1) * 2など)
